# Kiosk drink images (hero assets)

Place transparent PNG cutout images here. The touchscreen loads them by recipe name.

**Filename = exact recipe name + `.png`** (e.g. `Gin & Tonic.png`). Cape Codder now uses `Cape Codder.png`.

Required files for the current recipe set:

- Moscow Mule.png
- Gin & Tonic.png
- Whiskey & Cola.png
- Vodka Cranberry.png
- Whiskey Ginger.png
- Tequila Tonic.png
- Mexican Mule.png
- Cape Codder.png
- Gin & Juice.png
- Dirty Whiskey.png
- Long Beach Style.png

Missing images show a letter fallback (first letter of the drink name). No rectangular photo frame is applied; images are shown as standalone centered objects.
